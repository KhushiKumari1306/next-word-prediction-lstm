# 🚀 Next Word Prediction using LSTM

A Deep Learning project that predicts the next word in a given text sequence using an LSTM (Long Short-Term Memory) neural network built with TensorFlow/Keras. The project also includes a Streamlit web application for interactive predictions.

## 📌 Features

- 🔤 Predicts the next word based on an input sequence
- 🧠 Uses an LSTM-based Recurrent Neural Network (RNN)
- 📚 Trained on the Hamlet text dataset
- 🌐 Interactive web interface built with Streamlit
- 💾 Saves and loads the trained model and tokenizer

## 🛠️ Tech Stack

- Python
- TensorFlow / Keras
- NumPy
- Streamlit
- Pickle
